# text2mp3

## アプリケーションの概要

edge-ttsを利用したttsツールです。
下記を可能としています。

edge-ttsっているため下記が可能です。
- 流暢な読み上げ
- 高速の音声変換
- 字幕ファイルの作成
- 多言語での読み上げ

またtext2mp3として下記を可能に実装をしています。
- yamlでの変換文章の記載
- 辞書登録
- 行間の間を取る時間の調整
- 行ごとの音量・速度・音程の調整

## 利用することによってできること

### YouTubeなどの動画作成

辞書機能があることによって、
専門用語なども平仮名入力することなく
読み上げを行うことが可能となります。

字幕ファイルを作成していくれるため、
字幕をつけた動画の作成も容易に行えます。
また、動画編集時に話している内容に
合わせて映像を切り替えることが容易になります。

また、yamlのリスト形式で1つのファイルで
音声モデルを切り替えながら
音声変換が可能です。
そのため、会話形式の音声の作成も容易に行えます。

## 導入方法

本ソフトウェアでは、podman を利用しています。
images 自体はDocker Hubにあげているため、
下記コマンドの実施で利用可能となります。

### debian系 の場合

``` bash
sudo apt install podman
podman pull sabotagecla6/text2mp3
git clone https://github.com/sabotage-cla6/text2mp3.git
cd text2mp3
chmod u+x text2mp3
```

### Windowsの場合

実行スクリプトを作成後に記載します。

## 使い方

現時点、変換元のテキストファイル、辞書ファイル、変換後の音声ファイル、字幕ファイルはすべて同じディレクトリに保存している必要があります。

### debian系 の場合

1. git からcloneしたディレクトリに移動
1. 下記コマンドを実行
``` bash
./text2mp3 -v [音声/字幕を出力したいフォルダ] --input [変換元のテキストファイル名] -d [辞書ファイルのファイル名] -o [出力する音声ファイル名] -s [出力する字幕ファイル名]
```

手軽に試してみたい場合は下記のコマンドを入力してみてください。

``` bash
./text2mp3 -v ./ --input input.yaml -o output.mp3 -d dict.yaml -s output.srt
```
